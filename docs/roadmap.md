# Roadmap

## Milestone 0: Bootstrap

- Repository structure
- Public-ready documentation
- PWA dashboard shell
- Local development services
- CI for build, lint, and typecheck

## Milestone 1: Core Identity and Organization

- Tenant model
- User model
- Department hierarchy
- Roles and permissions
- PostgreSQL migrations and seed data
- Database health check API
- OIDC login path

## Milestone 2: Daily Groupware

- Schedule
- Facility reservation
- Notices and circulars
- Personal and team notifications
- Mobile navigation

## Milestone 3: Workflow and Documents

- Approval routes
- Application forms
- Document storage
- Version history
- Audit trail for approvals and file access

## Milestone 4: Operations

- Admin console
- Backup and restore
- Import/export
- Monitoring
- Security hardening checklist
